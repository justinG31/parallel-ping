#  parallel-ping

parallel-ping will take three websites as inputs, and ping each website 100 times.

# How to Run

Clone the repository in the terminal:
git clone https://github.com/justinG31/parallel-ping.git

Run the main program :
go run main.go


###  Websites to ping
    - google.com
    - youtube.com
    - bc.edu
    - gov.uk
